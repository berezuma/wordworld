Word World - Spelling AdventureA web-based spelling dictation application designed for progressive learning.FeaturesTwo Game Modes:Adventure Mode: Progression-based unlocking. Levels unlock sequentially as the user completes tests.Free Ride: Open access to all levels for practice.Progressive Difficulty: 10 levels categorized by linguistic patterns (Digraphs, Vowels, High-Frequency Words).Gamification:Point system (10 per word, 100 per level completion).Persistence via localStorage for scores and unlocked progress.Visual feedback using canvas-confetti.User Interface: Built with Tailwind CSS, featuring responsive design and high-contrast touch targets.Technical RequirementsAudio Assets: Must be located in a subdirectory named /audios/.File Naming: MP3 files must match the word list exactly (e.g., gift.mp3).Standardization: Words are normalized to lowercase and stripped of accents/diacritics for path resolution.Local SetupBrowsers restrict local file access via the file:// protocol due to CORS policies. To run the application locally:Place index.html in the root directory.Ensure the /audios/ folder exists with corresponding .mp3 files.Launch a local server:python3 -m http.server 8000
Navigate to http://localhost:8000.DeploymentCompatible with GitHub Pages. Ensure index.html is in the root of the repository.DependenciesTailwind CSS (via CDN)Canvas Confetti (via CDN)Google Fonts (Quicksand)
